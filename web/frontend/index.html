About CargoForge-C

CargoForge-C is a professional-grade maritime cargo loading simulator built in pure C99. It uses advanced 3D bin-packing algorithms and real naval architecture principles to optimize cargo placement while ensuring vessel stability.

Features:

  • 3D Guillotine bin-packing algorithm
  • Realistic stability calculations (GM, CG, IMO-aligned)
  • Cargo constraint handling (hazmat separation, weight limits)
  • Interactive 3D visualization
  • Progressive training challenges

Tech Stack: C99 backend, Python Flask API, Three.js visualization
