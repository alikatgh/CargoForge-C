# Interactive Demo

Try the CargoForge-C optimizer directly in your browser. This is a JavaScript port of the core C algorithms — no server required.

> **Note:** This demo uses the box-hull hydrostatic model. The full C binary supports hydrostatic table interpolation, free surface correction, longitudinal strength, and IMDG segregation. See [Stability Analysis](stability.md) for details.
